Gestão de Equipamentos - parte 2


A primeira versão do GesCol deu muito certo com aprovação massiva dos envolvidos em seu uso. Agora o desafio fica ainda mais sério e o objetivo é atender a companhia inteira, com seus mais de 20 Centros de Distribuição Logística, com um único sistema centralizado e servido via intranet.



Neste momento a decisão tomada foi de mudar de stack. Desta vez chegou a hora de aprender uma nova tecnologia, que nasceu para web e possui um framework incrível para tal: PHP e o poderosíssimo Laravel.



Este projeto ainda encontra-se em andamento, este "diário" será preenchido durante o caminho. Sinta-se convidado(a) a participar.
